对于矩阵$$A_(mxn)$$

|          | 基                                                           | 维度                          |
| -------- | ------------------------------------------------------------ | ----------------------------- |
| 列空间   | 主元所在的列构成一组基                                       | r，主元的个数，也就是矩阵的秩 |
| 零空间   | 自由变量所在的列构成一组基，一个求解的方法：例如对于拥有两个自由列的矩阵可以另自由变量的值为(1,0)(0,1) | n-r                           |
| 行空件   | 主元所在的行构成一组基                                       | r                             |
| 左零空间 | 自由行构成的一组基                                           | m-r                           |
